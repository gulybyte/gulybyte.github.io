Stack.

Serie - visão geral sobre estrutura de dados Java.

Como funciona?

  Stack é o que o próprio nome diz, uma pilha. Essa é uma estrutura clássica tipo LIFO (last-in first-out), o que significa que o ultimo a entrar é o primeira sair.

  Exemplo código:

var pilha = new Stack<Integer>();

pilha.push(10);
pilha.push(20);
pilha.push(30);

// exibindo o topo da pilha
System.out.println("Topo da pilha: " + pilha.peek());

// removendo elementos da pilha
System.out.println("Removendo elemento " + pilha.pop());

// exibindo o topo da pilha novamente
System.out.println("Topo da pilha: " + pilha.peek());
