Outras "queues"

  Vimos só ArrayDeque e PriorityQueue, mas tem outras mais principais de outras interfaces

  - BlockingQueue - abstrata um tipo de fila que espera que a fila
      não seja vazia ao recuperar um elemento e aguarda que o espaço fique
      disponível na fila ao armazenar um elemento;

  - BlockingDeque - é semelhante a BlockingQueue, mas para filas
      de extremidade dupla. É sub interface do BlockingQueue.
